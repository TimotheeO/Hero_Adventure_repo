HERO_ADVENTURE

Description :

Jeu de plateforme 2D rétro développé avec Godot.

Le projet propose une expérience inspirée des grands classiques du platformer rétro moderne avec :

exploration
collecte
speedrun
progression
boutique
combats

Le joueur traverse plusieurs niveaux remplis d’ennemis, de secrets et d’objets à collecter afin de débloquer de nouveaux contenus et améliorer ses performances.
